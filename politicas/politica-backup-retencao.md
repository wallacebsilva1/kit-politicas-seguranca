# Política de Backup e Retenção de Dados

## Objetivo
Garantir a disponibilidade e integridade das informações por meio de práticas básicas de backup e retenção.

## Abrangência
Aplica-se aos dados e arquivos considerados críticos para o negócio.

## Diretrizes de backup
- Realizar backup periódico dos dados críticos
- Manter versões históricas para possibilitar recuperação
- Garantir que os backups estejam protegidos contra acesso não autorizado

## Retenção
- Definir prazos de retenção conforme necessidade operacional e legal
- Evitar retenção excessiva de dados sensíveis

## Testes
- Realizar testes de restauração periodicamente para validar os backups

## Responsabilidades
- Área de TI: manutenção das rotinas de backup
- Gestores: identificação dos dados críticos de suas áreas
