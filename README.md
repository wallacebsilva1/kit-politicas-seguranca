# Kit Inicial de Políticas de Segurança da Informação

Este repositório contém um conjunto **simples e introdutório** de políticas básicas de segurança da informação, pensado para um cenário de **empresa pequena**.

O objetivo deste projeto é **praticar documentação**, entender conceitos fundamentais de segurança e demonstrar evolução prática na área de cibersegurança.

## Escopo
- Políticas básicas de segurança da informação
- Conceitos iniciais de controle e redução de riscos
- Documentação em nível iniciante

## Cenário de referência
- Empresa pequena (aprox. 25 colaboradores)
- Notebooks corporativos (Windows)
- E-mail e arquivos em nuvem (ex: Microsoft 365)
- Trabalho remoto ou híbrido básico

## Como usar
1. Leia as políticas disponíveis na pasta `/politicas`
2. Adapte nomes, responsáveis e ferramentas conforme a realidade da empresa
3. Utilize como base inicial para estabelecer controles mínimos de segurança

## Observação
Este é um **projeto de estudo em nível iniciante**.  
Não substitui consultoria especializada nem um programa completo de compliance.
